# Flux Studio Offer

Переработанный лендинг теперь целиком рендерится в React 18 (через ESM CDN), что позволяет управлять темизацией, мобильным меню и кастомным курсором в одном месте.

## Запуск

Никакой сборки не требуется — это прогрессивный статический бандл.

```bash
npx serve .
```

Либо просто откройте `index.html` в браузере.

## Что нового

- React-компоненты для шапки, блоков услуг, галереи и CTA.
- Хук темизации с поддержкой `prefers-color-scheme` и локального сохранения.
- Живой курсор и мобильное меню зарегистрированы как React-хуки.
- Новый раздел «Сигналы студии» с быстрыми программами.
