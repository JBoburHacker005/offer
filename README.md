# Flux Studio Offer

Минималистичный лендинг Flux Studio на чистом HTML/CSS + Vanilla JS.

## Запуск

Статический проект, поэтому достаточно открыть `index.html` в браузере.

Для локального сервера можно воспользоваться:

```bash
npx serve .
```

## Включено

- Шапка с темизацией (prefers-color-scheme + localStorage).
- Кастомный курсор и плавные состояния для ссылок/скролла.
- Блоки услуг, галерея, процесс, сигналы студии и CTA.
- Мобильное меню с оверлеем и блокировкой скролла.
